# ML Detection Object

Проект по детекции объектов на изображениях с использованием глубокого обучения.

##  Структура проекта

| Директория | Описание |
|------------|----------|
| `best_model/` |  Лучшее финальное решение |
| `Kirill_experiment/` |  Эксперименты Кирилла Усенкова |
| `Ruslan_experiements/` |  Эксперименты Руслана Козлова и Ксении Тихомировой |

##  Что внутри

- **Обучение моделей** — различные архитектуры и подходы
- **Ансамблирование** — комбинирование предсказаний нескольких моделей
- **Оптимизация** — подбор порогов, WBF, Soft-NMS
- **Аугментация** — расширение набора данных

- Руслан Козлов
